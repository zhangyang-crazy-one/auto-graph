|----------|-------------|
| normalized diagram -> Dagre wrapper | Untrusted or malformed numeric node sizes enter the external Dagre dependency. |
| Dagre label output -> DGE boxes | Third-party center coordinates are converted into DGE-owned top-left geometry. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Tampering | `src/layout/dagre.ts` | mitigate | Validate every node size with `Number.isFinite()` and non-negative checks; emit `layout.node-size.invalid` error diagnostics and omit unsafe boxes. |
| T-03-02 | Tampering | `src/layout/dagre.ts` | mitigate | Validate every edge source/target reference exists in the input node map; emit `layout.edge-reference.missing` and skip invalid Dagre edges. |
| T-03-03 | Denial of Service | `src/layout/dagre.ts` | accept | Dagre runtime is bounded to one call in this wrapper; heavier diagram-size limits are enforced in solver/routing plans. |
</threat_model>
